# Requisitos Funcionais #

Conforme definido nos [Casos de Uso Sistêmicos](ModelagemCasoDeUsoSistemico.md) a inteface Web do sistema deve fornecer as seguintes funcionalidades:

  * Amigo
    * Localizar
    * Exibir
    * Aceitar Convite
    * Manter
      * Adicionar
      * Remover

  * Fotos
    * Listar
    * Exibir
    * Manter
      * Adicionar
      * Remover

  * Vídeos
    * Listar
    * Reproduzir
    * Manter
      * Adicionar
      * Remover

  * Mural de Recados
    * Listar
    * Exibir
    * Manter
      * Adicionar
      * Remover
